# Arena of Valor Heroes Collection

## สรุปข้อมูลตัวละครในเกม Arena of Valor (ROV)

### จำนวนตัวละครทั้งหมด
- **ตัวละครปัจจุบัน**: 126 ตัว
- **ตัวละครที่กำลังจะเข้ามา**: 1 ตัว (Flowborn)
- **รวมทั้งหมด**: 127 ตัว

### ข้อมูลตัวละคร Flowborn
- **ชื่อ**: Flowborn (โฟลว์บอร์น)
- **ประเภท**: Marksman (แครี่)
- **กำหนดเข้าเกม**: 8 เมษายน 2025
- **ความสามารถพิเศษ**: สามารถเปลี่ยนสายได้ 4 สาย (Marksman, Mage, Support, Tank)
- **ลักษณะเด่น**: ฮีโร่ที่สามารถปรับเปลี่ยนรูปแบบการเล่นได้ตามสถานการณ์
